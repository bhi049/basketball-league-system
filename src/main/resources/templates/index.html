Welcome to Pro Ball Fantasy League!

Step into the world of fantasy basketball with this fictional league where you can manage teams, track players, and explore game standings.

About the Application

This application is a fictional basketball league built with the following technologies:

  • Spring Boot: Backend framework.
  • Maven: Dependency management tool.
  • Thymeleaf: Template engine for dynamic web pages.
  • MariaDB: Database for storing league data.
  • Rahti: Used for deployment.

The teams and players have been generated using AI tools to add a creative touch to the league. As this is a school project, the application is still in production. Future updates will include additional features and improvements.

Key Features

  • View the league standings, team details, and player stats.
  • Log in as a user to select your favorite team and player.
  • Log in as an admin to manage and edit the league in more detail, including teams and players.

How to Use

Navigate through the application using the menu at the top. Here's a quick guide:

  • Standings: View the current league standings.
  • Games: Check the scheduled and past games.
  • Teams: Explore team rosters and logos.
  • Players: Dive into player stats (coming soon) and profiles (coming soon).
